# Todo App

Een simpele todo applicatie gebouwd met PHP, MySQL en vanilla JavaScript.

## Features
- ✅ Todos toevoegen
- ✅ Todos bekijken
- ✅ Database opslag met MySQL

## Technologieën
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## Installatie
Zie [INSTALLATIE.md](docs/INSTALLATIE.md) voor uitgebreide installatie-instructies.

## Project Structuur
```
p3-startproject-thom/
├── public/         # Frontend bestanden
├── src/            # PHP classes
├── config/         # Database configuratie
└── docs/           # Documentatie
```

## Gebruik
1. Start XAMPP (Apache + MySQL)
2. Importeer database: `config/setup_database.sql`
3. Open: `http://localhost/school/github/p3-startproject-thom/public/`
